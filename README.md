# Be The Hero

WeekOminiStack 11.0 - Project to help ONGS to receive contacts and people that can help some incidents

### Prerequisites

What things you need to install the software and how to install them

```
Install node and npm 
```

## Installation

To run backend

```bash
npm install
npm start
```

To run frontend
```bash
npm install
npm start
```

To run mobile
```bash
yarn
yarn start
```

## Technologies
* Node
* ReactJS
* React Native
* Axios
* Express
* KnexJS
* SQlite3
